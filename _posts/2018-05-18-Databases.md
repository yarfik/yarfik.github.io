Database
========
Beberapa database yang di-support Django secara resmi (built-in) adalah:
- sqlite
- mysql
- postgresql
- oracle

3rd Party
=========
Selain keempat database di atas, juga ada vendor yang menyediakan library untuk database:
- SAP SQL Anywhere
- IBM DB2
- Microsoft SQL Server
- Firebird
- ODBC
